# PayPulse

**Fast. Secure. Nigerian.**  
The modern e-commerce platform built for speed and trust.


## Features
- Lightning-fast checkout
- Naira payments via Stripe
- Real-time cart with Zustand
- Dark mode + animations
- Built with React + Vite + Tailwind

## Tech Stack
- React 18
- Vite
- Tailwind CSS
- Supabase (Auth + DB)
- Stripe Payments
- Framer Motion
- Sonner Toasts
